DEBUG SCROLLING ISSUE

Cac buoc kiem tra:
1. Restart app va xem console output
2. Kiem tra "Đã tải X bài hát" - xem co bao nhieu bai
3. Neu co nhieu bai (>10) ma van khong scroll duoc, co the do:
   - CSS conflict
   - Layout structure sai
   - ListView bi fix height

Giai phap thu nghiem:
- Neu < 10 bai: them them bai de test scrolling
- Neu > 10 bai: kiem tra CSS va layout structure

Console output se cho biet exact so luong bai hat.
